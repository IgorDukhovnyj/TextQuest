Що вдалося зробити:
1) ігрова логіка
2) стартова сторінка
3) кнопки вибору
4) історія гравця
5) повернення до початку після програшу або перемоги
6) стилістика сторінок

Що треба доробити:
1) перевірку на порожнє поле імені гравця - наразі пропускає з іменем і без імені
2) додати кілька тестів
3) розширити лінійку кроків
4) додати деяку стилістику, можливо анімацію кнопок, бекграунд у стилі поточних кроків квесту
